# raza-fz.github.io

Personal website of **Mushahid Raza** — Program Management & Product Operations leader
specializing in AI-driven automation, workflow scaling, and marketplace operations.

🔗 **Live site:** https://raza-fz.github.io

## Built with
- Plain **HTML / CSS / JavaScript** — zero build step, zero dependencies
- Animated particle-network background (HTML canvas)
- Scroll-reveal animations & animated impact counters (IntersectionObserver)
- Fully responsive, dark "premium high-tech" theme
- Respects `prefers-reduced-motion`

## Files
| File | Purpose |
|------|---------|
| `index.html` | Page structure & content |
| `style.css`  | Theme, layout, animations |
| `script.js`  | Particle background, counters, scroll reveals, mobile nav |

## Run locally
Just open `index.html` in any browser — no server needed.
(Optional: `python3 -m http.server` then visit http://localhost:8000)

## Deploy
Hosted for free on **GitHub Pages** from the `main` branch (root).
Any push to `main` auto-publishes within ~1 minute.
